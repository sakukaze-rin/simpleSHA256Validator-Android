# Simple SHA256 Validator for Android

这是一个由 **Niriek_33** 和 **Gemini (Vibe Coding)** 共同开发的简单 Android SHA256 校验工具。

## 功能特点
- 📂 **文件选择**：快速选择手机内的任何文件。
- ⚡ **高效计算**：自动计算文件的 SHA256 哈希值。
- 📋 **点击复制**：计算结果支持一键复制到剪贴板。
- ✅ **状态对比**：支持粘贴原始哈希值进行自动比对。
